### Requirements ###

* [Docker v17 or higher](https://www.docker.com/community-edition)
* [Docker Compose v1.13 or higher](https://docs.docker.com/compose/)
* [Bash Shell v4 or higher](https://www.gnu.org/software/bash/)

```
#!bash

bash dev.sh
```
